*IDE used : IntelliJ , MySQL workbench

*Used Technologies and Tools:
1- Spring boot
2- Maven
3-Spring web 
4-Springboot JPA
5-Rest Controller

*Steps to run applicaton:
1- please run scripts in "\src\main\resources\scripts\chatApp.sql" 
2- To send message from Postman:
	2.1- please visit "http://localhost:8082/message/loginwithmsg"
	2.2- Please pass user name and password parameters (Keys : "userName", "password") 
	--values must be inserted first in database or select values from exist data.
3-To get Statestics for a conversation of specific user
	3.1-visit :"http://localhost:8082/message/countUserChat"
	3.2-pass user name parameter (key : "userName")
4-To get statestics for all chats : "http://localhost:8082/message/countAllUsersChats"
